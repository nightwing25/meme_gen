# meme_gen
random meme generator
if you want to play around with the code process below 


##-GETTING STARTED:
1.HEAD OVER TO [apileague](https://apileague.com/apis/search-memes-api/) 
2.LOGIN AND TO GET API KEY FROM YOUR CONSOLE 
3.copy api key

##-INSTALLATION(debian/ubuntu):
1.apt install update && apt install upgrade
2.git clone github.com/nightwing25/meme_gen.git
3.pip install -r requirements.txt
4.place api key MemeFinder.py api_key section for now save and quit 
5.python memeFinder.py
